# VOID — Zero-Waste Fashion Website

A premium, monochrome fashion portfolio website with red accent design.

## Project Structure

```
zero-waste-fashion/
├── index.html        Home Page
├── about.html        About Page
├── story.html        Visual Story (6 images)
├── garments.html     Collection (8 garments)
├── zero-waste.html   Process Page
├── designer.html     Designer Page
├── contact.html      Contact Page
├── css/styles.css    All styles + animations
├── js/script.js      Interactivity
└── assets/
    ├── images/
    │   ├── garments/     → g1.jpg ... g8.jpg
    │   ├── process/      → p1.jpg ... p4.jpg
    │   ├── story/        → story1.jpg ... story6.jpg
    │   └── designer/     → portrait.jpg, working.jpg
    └── videos/           → hero.mp4
```

## Adding Your Images

Replace the placeholder divs by adding real `<img>` tags, or place your images in the asset folders and update the `src` attributes.
